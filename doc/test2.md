# 📘 GitHub Training – Session 1  
## Introduction to Git & GitHub

---

## 1. Learning Objectives
By the end of this session, learners will be able to:
- Understand what **Git** and **GitHub** are
- Explain why version control is important
- Create a GitHub account
- Create a repository
- Upload (commit) files to GitHub

---

## 2. What is Git?

### 🔹 Definition
- **Git** is a **version control system (VCS)**
- It tracks changes in files over time

### 🔹 Why Git is important
- Keeps history of changes
- Allows reverting to previous versions
- Enables collaboration without conflicts

### 🔹 Example
> Like “Track Changes” in Word, but more powerful and designed for managing files and code

---

## 3. What is GitHub?

### 🔹 Definition
- **GitHub** is a **cloud-based platform** that hosts Git repositories

### 🔹 Key Features
- Remote storage of projects
- Collaboration with others
- Backup and version tracking
- Open-source sharing

### 🔹 Real-world Use
- Used by developers, companies, and researchers worldwide

---

## 4. Git vs GitHub

| Git | GitHub |
|-----|--------|
| Tool (software) | Platform (service) |
| Runs locally | Runs on the cloud |
| Tracks changes | Hosts repositories |

---

## 5. Key Concepts

### 📌 Repository (Repo)
- A project folder tracked by Git

### 📌 Commit
- A “snapshot” of changes

### 📌 Branch (Preview)
- A separate line of development

### 📌 Clone (Preview)
- Copy a repository to your computer

---

## 6. Creating a GitHub Account

### Step-by-Step
1. Go to https://github.com
2. Click **Sign up**
3. Enter email, password, username
4. Verify your account
5. Complete setup

---

## 7. Creating Your First Repository

### Steps
1. Click **New Repository**
2. Enter repository name  
   - Example: `my-first-project`
3. Add description (optional)
4. Choose:
   - Public / Private
5. Click **Create Repository**

---

## 8. Uploading Files (Web Method)

### Steps
1. Open your repository
2. Click **Add file → Upload files**
3. Drag and drop files
4. Write a commit message  
   - Example: `Initial upload`
5. Click **Commit changes**
